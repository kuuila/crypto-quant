## Layer 1: 预判层（决策支持）

### 外部工具集成

| 工具 | 用途 | 数据 |
|------|------|------|
| **TradingView** | 技术分析 | 图表、指标、形态 |
| **Glassnode** | 链上数据 | 交易所流入流出、持币分布 |
| **CoinGlass** | 持仓数据 | 未平仓合约、资金费率 |
| **Alternative.me** | 情绪指标 | Fear & Greed Index |
| **CoinMarketCal** | 事件日历 | 利好利空事件 |

### 预判报告模板

```
📊 BTC/USDT 综合预判报告
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📈 技术面 (TradingView)
  趋势: 上涨
  关键位: 支撑 68k / 阻力 72k
  信号: 突破中轨，可能上攻

🔗 链上数据 (Glassnode)
  交易所流出: -15,000 BTC (利好)
  长期持有者: 增加 2.1% (利好)

📊 持仓数据 (CoinGlass)
  未平仓合约: $18.5B (高位)
  资金费率: +0.01% (中性偏多)

😨 市场情绪 (Fear & Greed)
  指数: 65/100 (贪婪)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎯 综合判断: 偏多，但高位谨慎
💡 建议: 网格上移或轻仓做多
⚠️ 风险: 如跌破 68k 则转空
```

## Layer 2: 监控层（信号触发）

### 监控指标

| 监控项 | 触发条件 | 动作 |
|--------|----------|------|
| **价格突破** | 跌破支撑/突破阻力 | 推送 + 可能止损 |
| **成交量异动** | 量比 > 3 倍 | 推送预警 |
| **资金费率** | 超过 ±0.05% | 可能套利机会 |
| **大额转账** | 链上 > 1000 BTC | 推送预警 |
| **持仓变化** | OI 短时增减 > 10% | 市场转折信号 |

## Layer 3: 执行层（策略执行）

### 策略选择决策树

```
预判结果
   ├─ 看涨 + 单边趋势 → 趋势跟踪策略
   ├─ 看跌 + 单边趋势 → 空仓或做空
   ├─ 震荡 + 波动适中 → 网格交易
   ├─ 高波动 + 不确定 → 缩小仓位观望
   └─ 有套利空间 → 期现套利
```

### 执行模式

1. **全自动模式**: 根据预判自动选择策略执行
2. **半自动模式**: 推送预判和信号，人工确认后执行
3. **信号模式**: 仅推送信号，完全人工操作

## 工作流

```
每日 8:00 → 生成预判报告 → 推送 Telegram
           ↓
实时监控 → 触发信号 → 推送 + 可选自动执行
           ↓
策略执行 → 成交/止损 → 推送结果
           ↓
每日复盘 → 优化参数 → 更新策略
```
