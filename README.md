## 核心功能

### 📊 Layer 1: 预判层
- **技术分析**: MA、RSI、布林带、ATR
- **情绪指数**: Fear & Greed Index
- **持仓数据**: 资金费率、多空比、OI
- **综合报告**: 自动生成交易建议

### 🔍 Layer 2: 监控层
- 价格涨跌幅监控
- 成交量异常检测
- 资金费率告警
- 多空比极端预警
- 情绪指数监控
- **Telegram 实时推送**

### ⚡ Layer 3: 执行层
- 网格交易策略
- 趋势跟踪策略
- 策略自动切换
- 风控联动

## 快速开始

```bash
# 克隆
git clone https://github.com/kuuila/crypto-quant.git
cd crypto-quant

# 安装依赖
python3 -m venv venv
source venv/bin/activate
pip install ccxt pandas requests python-telegram-bot

# 配置 API
cp .env.example .env
# 填入 Binance API Key

# 运行
python bot/quant_system.py
```

## 监控规则（可配置）

| 监控项 | 阈值 | 动作 |
|--------|------|------|
| 价格涨幅 | > 3% | 预警 |
| 价格跌幅 | > 3% | 预警 |
| 成交量 | > 3x | 预警 |
| 资金费率 | > 0.05% | 预警 |
| 多空比 | > 70% | 预警 |
| Fear & Greed | < 20 或 > 85 | 预警 |

---

## 策略决策

```
预判结果 → 自动选择策略:
- 单边上涨 → 趋势跟踪
- 单边下跌 → 空仓/做空
- 震荡整理 → 网格交易
- 高波动   → 缩小仓位
- 方向不明 → 观望
```
