## 技术栈选型

| 组件 | 选择 | 理由 |
|------|------|------|
| 语言 | Python 3.11 | 生态最丰富 |
| 交易所API | CCXT | 统一接口，支持 100+ 交易所 |
| 框架 | freqtrade (参考) | 开源成熟，策略丰富 |
| 数据 | CCXT + 自建缓存 | 够用 |
| 回测 | backtesting.py | 轻量易用 |
| 部署 | Docker | 跨平台一致性 |
| 监控 | Telegram Bot | 实时推送 |

## 策略方向

**一期目标**：网格交易（Grid Trading）
- 理由：适合震荡行情，逻辑简单，风险可控
- 资金利用率中等
- 可自动运行，无需盯盘

**二期目标**：资金费率套利
- 理由：低风险，收益稳定
- 需要多交易所账号
